• 1. Git Diff

    • Première chose à faire, vous rendre dans votre dossier 'coding-school-12/GIT'
    • Y créer un dossier exo_git_final qui sera votre projet d'apprentissage git final
    • Initialiser ce projet avec git
    • Créer un fichier index.html
    • Rajouter la structure html de base
    • Rajouter un h1 avec comme contenu hello world
    • Commiter avec comme nom "h1 = hello world"
    • Créer une nouvelle branch votreprenom
    • Mettez vous sur cette nouvelle branch
    • Changez le contenu de h1 "hello world" par "Goodbye"
    • Commiter avec comme nom "h1 = Goodbye"
    • Retourner sur la branch master
    • Faites un git diff master votreprenom

    • 2. Git Merge

      Vous venez de faire un git diff qui vous permet de voir la différence entre deux branch.

      La version de prénom vous convient,dans se cas vous allez merger prénom sur master,c'est à dire écraser la version de master par la version de prénom

    • en étant sur la branch master faites : git merge votreprenom

    • 3. Les Conflits

      Imaginons le cas ou plusieurs développeur sur le meme projet venaient à faire une modification au même endroit, comment les choses seraient elles gérées ?

    • Changez le contenu de votre h1 par "ceci est la branch master"
    • Faite un commit "changement du contenu du h1 par 'ceci est la branch master'"
    • Rendez vous sur la branch prénom
    • Changez le contenu du h1 par "ceci est la branch prénom"
    • Faite un commit "changement du contenu du h1 par 'ceci est la branch votreprenom'"
    • Retournez sur master
    • Mergez

      • Vous devriez obtenir un résultat comme celui ci

    • Acceptez les donnés entrantes

      • Vous venez de résoudre un conflit en choisissant la version de prénom plus tot que celle de master
